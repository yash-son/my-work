In html and css colors can be specified using the rgb,hex,hsl,rgba and hsla values.

RGB

RGB color represents red,green and blue light sources.

In html color can be specified in this format rgb red,green,blue

For example, rgb(255, 0, 0) is displayed as red, because red is set to its highest value (255) and the others are set to 0.

To display black, set all color parameters to 0, like this: rgb(0, 0, 0).To display white, set all color parameters to 255, like this: rgb(255, 255, 255).

The shades of grey are usally specified by using the same value for all three light sources.

Rgba color value

It is an extension of rgb with additional alpha channel to define the opacity of the color like this rgba(red, green, blue, alpha).

The alpha parameter value 0.0 specifies fully transparent and 1.0 specifies not transparent at all.

HEX color value

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.

The css color can be specified using the hexadecimal value using this format:
#rrggbb

to specifies red color in hex format we use #ff0000 as we have full red ff , no green and blue 00.

To specify black set all the values to 00.

To specify the white color set all the values to their max ff.

By using the equal values of all three light sources we get the shades of gray.

Three digit hex format

Its a shorthand for 6 digit hex format.

The 3-digit hex code has the following form:#rgb,where the r,g,b represents red,green and blue sourec between 0 and f.

The 3-digit hex code can only be used when both the values (RR, GG, and BB) are the same for each component. So, if we have #ff00cc, it can be written like this: #f0c.

CSS hsl colors

Hsl stands for hue,saturation and lightness.

CSS, a color can be specified using hue, saturation, and lightness (HSL) in the form:
hsl(hue, saturation, lightness)

Hue is a degree of color wheel from 0 to 360.0 is red,120 is green and 240 is blue.

Saturation means how much color.0 means shade of grey and 100 means full color.

Lighness is also a percenatage 0 means black,50% is neither black nor white and 100 is white.

shades of grey are often get by setting both hue and saturation at zero.

hsla format

It it an extension of hsl with extra of alpha channel for opacity values from 0.0 to 1.0. 0.0 means fully transparent while 1.0 means fully opaque.
