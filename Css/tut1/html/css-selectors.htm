CSS selectors

Css selectors are used to find or select the html elements you want to style.

We can devide the css selectors in these five categories.

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)
  • Attribute selectors (select elements based on an attribute or attribute value)

CSS slement selector

Element selects the html element based on its name.

Css id selector

The css id selectors uses id attribute value of element to style it.

To select an element with start by # followed by its id value to style it.

Id is unique in a html document so its is used to style a single particular element.

An id name cannot start with a number.

The css class selector

the css class selects elements with specific class value.

To selects the elements with a particular class write its name after .

class can be used to identify multiple elements.

Note:A class name cannot start with a number.

css universal selector

The universal selector * selects all the elements in a body of html document.

grouping selectors

grouping selector group all the selectors with the same styling to minimize code.
