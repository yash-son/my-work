Head element

The HTML head> element is a container for the following elements: title>, style>, meta>, link>, script>, and base>.

The element is a container for metadata (data about data) and is placed between the html> tag and the body> tag.

htMl meta tag

The element is typically used to specify the character set, page description, keywords, author of the document, and viewport settings.

Meta data is not dsipalyed by the browser but used by browsers on how to load the page ,locate it in the web-server ,by search engines and other web services.

Setting the viewport

Viewport the user visible area of the web page.

We should always include the meta viewport in our html code.This helps browser in setting the wdth and intial scaling of the web-page.

The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device).

The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.

Base tag

Base tag is used to define a base url or target for all relative url's in a page.

There can only be one single base element in a html document.

using base tag we can set the common default target for all the links i the webpage.
