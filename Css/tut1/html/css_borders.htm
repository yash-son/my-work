CSS border

CSS border property allows you too specify the style,width and color of the border.

CSS border-style property

It specifies what kind of border to display.

The followinf values are allowed:

  • dotted:Defines a dotted border
  • dashed: defines a dashed boreder
  • solid: Defines a solid border
  • double: Defines a double border
  • groove: Defines a 3D grooved border. The effect depends on the border-color value
  • ridge: Defines a 3D ridged border. The effect depends on the border-color value
  • inset: Defines a 3D inset border. The effect depends on the border-color value
  • outset: Defines a 3D outset border. The effect depends on the border-color value
  • none: Defines no border
  • hidden: Defines a hidden border
  • Mixed: Its a mix of four different border types in four different sides.

Note: The other css properties would have no effect if border-style is not specified.

CSS boder width property

The border width prperty specifies the width of the four sides.The sequence in which different side should be given is top right bottom and left
